# 📱 Mobile Application Development — PKCERT Internship

This repository documents my weekly progress and hands-on projects completed
during my **Mobile Application Development internship at PKCERT**. Each week's
work builds on the previous one, covering everything from Flutter environment
setup and Dart fundamentals to building fully functional cross-platform mobile
applications using **Flutter** and **Dart**.

## 🎯 About This Internship

As part of this internship, I am gaining practical experience in mobile app
development using Google's Flutter framework — learning industry-standard
tools, workflows, and best practices for building responsive, high-performance
applications for Android and iOS from a single codebase.

## 🛠️ Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **IDE:** Visual Studio Code / Android Studio
- **Version Control:** Git & GitHub
